Narrative Rethink: Journey, Not Museum

The current design treats rooms as exhibit halls. Here's what it should be instead.

Before: Museum Mode

  • Rooms organized by category (systems, AI, visual)
  • Press E on colored square, read project description
  • No story connecting rooms
  • Interactables are basically LinkedIn bullet points
  • Spider companion says "ooh this room is cool"
  • No reason to visit rooms in any order
  • Visitor thinks: "neat demo" and closes tab

After: Journey Mode

  • Rooms are chapters of Eren's builder story
  • Environmental storytelling through tile art
  • Spider companion narrates, reacts, has personality
  • Each room has a MOMENT — not just info dumps
  • Dialogue tells WHY things were built, not just WHAT
  • Rooms change/react as you explore (tiles swap, sprites appear)
  • Visitor thinks: "this person is insane, I need to hire/follow them"

Proposed Narrative Arc

Each room is a chapter. The spider companion (Arachne) guides you through Eren's story.

Ch. 0
The Web (Welcome + Nexus merged)
You wake up in a dark void. A spider web materializes around you. The spider speaks:
"You found the web. Good. Let me show you what Eren built."
The Nexus hub has 4 web strands leading to doors. Each strand PULSES (tile color oscillates) to draw you in. The web center interactable reveals: "This game? 389 kilobytes. The engine running it? Eren built that too. Keep walking."
Ch. 1
The Forge — "I wanted to understand how things actually work"
Story beat: Eren's move from Python/JS into Rust and systems programming.
The room starts cold (stone). As you walk deeper, lava appears — the fire of going low-level.
Arachne anvil: "Everyone uses Unity. 300MB runtime for a 2D game. I thought: what if I just... didn't?"
Synapse furnace: "ML models shouldn't need a data center. 18M params running in your browser. On a phone. Fight me."
sqld monument: "My AI agent wrote a database engine. 26,500 lines of Rust. I watched it happen. Then I had to prove in court that I owned it."
The sqld story (legal battle) is a MOMENT — multi-step dialogue that builds tension.
Ch. 2
The Hive — "What if the machine could build WITH me?"
Story beat: The AttoCode journey — from tutorial to framework to swarm system.
Room feels alive: multiple "agent" sprites (small colored squares) drift around the room autonomously.
The Origin terminal: "Started as a 26-lesson TypeScript course. Teach yourself AI coding agents from scratch."
The Swarm core: "Then I gave the agents a goal: build a database engine. They did. Then a programming language."
Spider Chat node: "Conversations shouldn't be linear. What if chat branched like git? I built that."
The drifting agent sprites are the "wow" — tiny squares moving in patterns, showing the swarm concept visually.
Ch. 3
The Lens — "Teaching machines to see"
Story beat: Visual AI work — from face recognition to browser automation.
Ice crystal room represents precision/clarity of computer vision.
Visionbot station: "Browsers are the universal interface. So I built agents that can see and use them. Like humans, but faster."
SAM-CLIP display: "Point at something. Describe what you want. The pipeline figures out the rest. Three models, one sentence."
The Timeline wall: A row of interactables showing progression: Morse blink detection > face API > geo spy > SAM-CLIP > Visionbot.
"The first thing I built detected Morse code from eye blinks. Then faces. Then entire web pages."
Ch. 4
The Lab — "Break things on purpose, learn faster"
Story beat: Research/experimentation mindset. Crucible, ML experiments, weird projects.
Mossy, overgrown feel — experiments that grew wild. Deep water pools = depth of exploration.
Crucible altar: "OpenAI ran a competition. I built a system that designs its own ML experiments. LLMs proposing hypotheses, GPUs running them. 865 tests."
Antelligence terrarium: "I simulated ant colonies to see if intelligence could emerge from simple rules. It did. Then I got distracted by something shinier."
The Shelf: Multiple small interactables — one-liner descriptions of side projects. Rewards thorough explorers. "AI Tamagotchi. Auto cover letters. OCR tools. I build things the way other people doomscroll."
Ch. 5
Eren's Room — "Human, talking to machines"
Story beat: The person behind the code. Not a project room — a personal space.
Warm, cozy. The only room with grass and water (a garden). Feels like coming home.
The Desk: "M.Sc. in Intelligent Systems. Based in Berlin. Mass producer of open source. Talks to machines for a living."
The Window: "Berlin. Good coffee, fast internet, no small talk. Perfect for building things."
The Bookshelf: "blog.akbuluteren.com — where I write about the things I break and occasionally fix."
Hidden interaction (3x on the rug): Floor opens to The Void. "Oh. You found it."
Ch. 6
The Void — "You're inside the proof"
Story beat: The meta-revelation. The game explains itself.
Nearly empty. Lava border. One interactable at center.
The Core: Multi-step dialogue that builds:
"This game is 389 kilobytes."
"The engine running it was built from scratch. In Rust. By me."
"You just played a portfolio built on the portfolio piece."
"That's the kind of thing I build. Want to build something together?"
"github.com/eren23 | blog.akbuluteren.com"
After this dialogue, the room's tiles change — the void fills with web strands connecting to all rooms. The web is complete.

What Changes Visually

More environmental storytelling: Tile art that creates scenes, not just walls+floor. The Forge has an actual anvil shape in dark brick. The Hive has hex-pattern tiles. The Lens has a viewport/eye shape in ice.

Dynamic elements: Agent sprites moving in The Hive. Tiles changing after interactions (Void fills with web). Silk thread counter that means something (unlocks hidden door).

More interactables per room: 5-8 per room instead of 2-4. Environmental objects (cracks in walls, notes on floors, hidden spots) not just project terminals.

The spider companion actually talks: Different reaction per room. Comments on things. Has opinions. "I helped build this one" vs "Eren stayed up 3 days straight on this."

Journey progression: Each room unlocks context for the next. The Forge makes more sense after hearing about the Hive. The Void only lands if you've seen the full picture.

The hidden door trigger: Interact with the rug 3 times in Eren's Room. No hint. Explorers find it. OR collect all 8 silk threads. Two paths to the secret.
