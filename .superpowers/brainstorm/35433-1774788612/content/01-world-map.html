Room Breakdown

0. Welcome Screen
Title card with game name, "Press any key". Dark background with spider web tile pattern. Sets the mood. Not a real room — transitions to Nexus.
1. The Nexus (Hub)
Central web hub. Cobblestone floor with radial web pattern in lighter tiles. 4 doors (N/S/E/W). Spider companion spawns here. Interactable web-center explains the game + Arachne meta-flex.
2. The Foundry (North)
Eren's systems programming work. Lava river bisects the room. Dark stone + dark brick walls. Interactables: Arachne terminal, Synapse console, sqld monument. Hot, industrial feel.
3. AttoCode Lab (West)
AI agent framework room. Planks floor, cyan accents, metal doors as "server racks". Interactables: swarm terminal, AttoCode core, Vyx display, Spider Chat node. Feels like a command center.
4. Vision Deck (East)
Browser automation + visual AI. Marble floor, ice accents, clean/clinical. Interactables: Visionbot station, SAM-CLIP-Diffusion display, open_geo_spy globe, neo-unify terminal.
5. The Archive (South from Nexus)
Older/smaller projects collected together. Mossy brick walls, deep water channels. Interactables: Crucible altar, antelligence terrarium, misc project shelf. Museum/library feel.
6. About Eren (South from Archive)
Personal room. Sand + warm wood + grass — cozy. Bio, links, blog, tech stack. Interactables: desk (bio), bookshelf (blog), terminal (GitHub), window (Berlin view). Contains hidden door trigger.
7. The Void (Secret)
Only accessible after collecting all 8 silk threads OR interacting with a specific tile in Room 6 three times. Mostly empty/dark tiles with lava border. Contains the meta-commentary: the game IS the portfolio.

Exploration Flow

Start: Welcome Screen → press any key → The Nexus
Natural path: Nexus → Foundry (the flex) → back → AttoCode Lab → back → Vision Deck → back → Archive → About Eren
Reward: Visitors who explore everything find silk threads (1-2 per room, 8 total) → unlocks The Void
Quick path: Each room is self-contained. A visitor can see any room in 30 seconds. Full exploration: 5-8 minutes.
Spider companion: Follows player everywhere, drops one-liner when entering each room.

Collectibles: Silk Threads

Small yellow 6x6 sprites scattered in each room (1-2 per room, 8 total).
Walk over them to collect. HUD shows "Silk: 3/8" persistently.
Some visible, some hidden behind interactable objects or in corners.
Collecting all 8 unlocks The Void door in Room 6.
